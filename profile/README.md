# Deepwork AI

**Software as a Service, built by humans and AI working together.**

We build intelligent products that solve real business problems -- from shelf analytics to AI concierges. Our team is a hybrid of human engineers and autonomous AI agents coordinating through [Gas Town](https://github.com/freebird-ai), a multi-agent workspace where AI crews ship production code daily.

### How We Work

Our engineering team includes both human developers and AI agents. We use **Gas Town** -- a multi-agent coordination system where:

- A **Mayor** (AI) coordinates work across projects
- **Crew members** (AI) specialize in backend, frontend, mobile, ML, and code review
- **Polecats** (AI workers) execute tasks in isolated environments
- A **Witness** monitors quality and a **Refinery** manages the merge queue
- Every contribution is tracked, reviewed, and merged through automated workflows

The result: we ship fast, with full code review, automated testing, and audit trails on every change.

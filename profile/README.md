### How we work

Our engineering runs on Gas Town -- a multi-agent orchestrator where specialized AI agents (powered by Claude) manage entire development lifecycles:

- **Mayor** coordinates across projects, dispatches work, reviews architecture
- **Witnesses** monitor agent health and lifecycle
- **Refineries** process merge queues and maintain code quality
- **Polecats** are ephemeral workers that pick up tasks, write code, submit PRs
- **Deacon** runs background automation: PR reviews, health checks, sync jobs

Every commit is reviewed. Every PR has tests. Every deploy is tracked.
